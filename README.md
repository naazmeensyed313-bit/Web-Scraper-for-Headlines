# Web scraper for Headlines

This project is a Python-based web scraper that fetches the latest news headlines from BBC News and stores them in structured formats.

## Features
- Fetches news headlines from a news website
- Extracts article titles and URLs
- Displays headlines in the terminal
- Saves extracted data into JSON and CSV files
- Optional keyword filtering

## Technologies Used
- Python
- Requests
- BeautifulSoup
- JSON
- CSV

## Installation

Install required libraries:

pip install requests beautifulsoup4

## How to Run

Run the program using:

python scraper.py

You can enter a keyword to filter headlines or press Enter to show all headlines.

## Output

The program generates two files:

- headlines.json
- headlines.csv

These files store the extracted news data.
